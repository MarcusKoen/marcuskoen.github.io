# Specifications
1. 16 8-bit registers
2. 4096 bits total memory
3. PC should be 16-bit
4. The original implementation of the Chip-8 language includes 36 different instructions, including math, graphics, and flow control functions.
5. All instructions are 2 bytes long and are stored most-significant-byte first. In memory, the first byte of each instruction should be located at an even addresses.

Note to self, cannot switch in class declaration, only in functions.

# Stack Pointer Convention
sp points to next free slot
initial state sp=0;
stack is empty
next push goes to stack[0]

# Shifting
```
| Field          | Mask   | Shift |
| -------------- | ------ | ----- |
| X (register)   | 0x0F00 | >> 8  |
| Y (register)   | 0x00F0 | >> 4  |
| N (low nibble) | 0x000F | none  |
| KK (byte)      | 0x00FF | none  |
| NNN (address)  | 0x0FFF | none  |
```
# Opcodes
```
3.1 - Standard Chip-8 Instructions
      00E0 - CLS
      00EE - RET
      0nnn - SYS addr
      1nnn - JP addr
      2nnn - CALL addr
      3xkk - SE Vx, byte
      4xkk - SNE Vx, byte
      5xy0 - SE Vx, Vy
      6xkk - LD Vx, byte
      7xkk - ADD Vx, byte
      8xy0 - LD Vx, Vy
      8xy1 - OR Vx, Vy
      8xy2 - AND Vx, Vy
      8xy3 - XOR Vx, Vy
      8xy4 - ADD Vx, Vy
      8xy5 - SUB Vx, Vy
      8xy6 - SHR Vx {, Vy}
      8xy7 - SUBN Vx, Vy
      8xyE - SHL Vx {, Vy}
      9xy0 - SNE Vx, Vy
      Annn - LD I, addr
      Bnnn - JP V0, addr
      Cxkk - RND Vx, byte
      Dxyn - DRW Vx, Vy, nibble
      Ex9E - SKP Vx
      ExA1 - SKNP Vx
      Fx07 - LD Vx, DT
      Fx0A - LD Vx, K
      Fx15 - LD DT, Vx
      Fx18 - LD ST, Vx
      Fx1E - ADD I, Vx
      Fx29 - LD F, Vx
      Fx33 - LD B, Vx
      Fx55 - LD [I], Vx
      Fx65 - LD Vx, [I]

nnn or addr - A 12-bit value, the lowest 12 bits of the instruction
n or nibble - A 4-bit value, the lowest 4 bits of the instruction
x - A 4-bit value, the lower 4 bits of the high byte of the instruction
y - A 4-bit value, the upper 4 bits of the low byte of the instruction
kk or byte - An 8-bit value, the lowest 8 bits of the instruction

```
